# PM-SP 2026 — Arena do Recruta

Site gamificado de estudos para o concurso PM-SP Aluno-Soldado 2026.

## Funcionalidades

- Dashboard com missão inteligente.
- Checklist diário TDAH.
- Modo dia ruim de 10 minutos.
- Sistema de XP, nível, patente, medalhas e sequência.
- Banco com 350 questões.
- Revisão de questões erradas.
- Flashcards.
- Backup/exportação/importação do progresso.
- Simulado completo oficial com 60 questões.
- Redação com temas e checklist.
- TAF com plano e registro de desempenho.
- PWA instalável no celular.

## Banco de questões

Matemática: 100
Conhecimentos Gerais: 80
Informática: 35
